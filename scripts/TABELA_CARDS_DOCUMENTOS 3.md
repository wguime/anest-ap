## 🔄 Como Vincular Novos Documentos

Para vincular novos documentos aos cards, edite o arquivo:
- `App/scripts/vincular-documentos-cards.js`

Adicione uma nova entrada no array `documentosParaVincular` seguindo o padrão:

```javascript
{
    arquivo: 'Documentos/caminho/arquivo.pdf',
    collection: 'nome_da_collection',
    storagePath: 'nome_da_pasta',
    titulo: 'Título do Documento',
    descricao: 'Descrição do documento'
}
```

Depois execute:
```bash
node scripts/vincular-documentos-cards.js
```
